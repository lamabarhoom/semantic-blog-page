# Blog Article Page - API Guide

## Project Overview
This is a **Week 3 assignment** project that creates a comprehensive blog article page about APIs (Application Programming Interfaces) in Arabic. The article is designed as an educational resource for beginners to understand the fundamentals of APIs and their role in modern web applications.

## Article Details
- **Title**: عالم الـ API — شرح شامل للمبتدئين (API World - A Comprehensive Guide for Beginners)
- **Author**: لمى (Lama)
- **Publication Date**: May 11, 2026
- **Language**: Arabic (RTL - Right-to-Left)

## Features
- **Responsive Design**: Mobile-friendly layout with viewport meta tag
- **Navigation Menu**: Quick links to all main sections
- **Semantic HTML**: Proper semantic structure using header, main, article, and section elements
- **Multimedia Content**: 
  - Placeholder images
  - Video player
  - Audio podcast player
  - Embedded YouTube video
- **Accessibility**: Proper use of ARIA labels and semantic HTML

## Page Sections
1. **المقدمة (Introduction)** - Overview of APIs and their importance
2. **ما هو الـ API؟ (What is API?)** - Definition and explanation with blockquote
3. **الوسائط المتعددة (Multimedia)** - Educational videos, podcasts, and embedded content
4. **الأسئلة الشائعة (FAQ)** - Frequently asked questions
5. **المراجع (References)** - Source citations
6. **تواصل معنا (Contact Us)** - Contact information

## File Structure
```
├── index.html          # Main blog article page
└── README.md           # Project documentation
```

## Usage
Simply open `index.html` in your web browser to view the blog article page.

## Technologies Used

- Right-to-Left (RTL) text direction for Arabic content
